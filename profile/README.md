SimplexLab is an open-source community whose mission is to develop humanity’s collective intelligence.

We believe intelligence is an emergent property: new dimensions of intelligence arise when intelligent systems are combined appropriately.

Our main goal is to design and improve tools that enable humans to combine their intelligence more effectively, and to build a community around them.

---

### Projects:

- [TorchJD](https://github.com/SimplexLab/TorchJD/): TorchJD is a library extending autograd to enable
  [Jacobian descent](https://arxiv.org/pdf/2406.16232) with PyTorch. The full documentation is available at [torchjd.org](https://torchjd.org),
  with several usage examples.
